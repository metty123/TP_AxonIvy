When check out this project from SVN, you may have error at this file java file due to missing library
../AdminUI/src/ch/ivyteam/ivy/admin/tool/RoleService.java
To solve it, add this jar lib(lib/ivy/ch.ivyteam.ivy.user.role_xxx.jar) from Axon Engine file to this project
